              # 1. 安装前注意事项

              # 电脑配置

              建议运行内存 4G 及以上，系统 Win10 及以上。

              # 杀毒防火墙 - 非常重要

              建议卸载所有杀毒、管家类软件，比如2345、腾讯电脑管家、360安全卫士、360杀毒、金山毒霸；

              然后安装火绒安全软件open in new window

              万一没处理被杀毒删了，可以再次双击安装包修复，点我查看详细步骤

              # 2. 软件下载安装

              下载地址：https://www.faqun.cn/faqunopen in new window

              下载红包多多软件、配套微信，双击即可傻瓜式安装！配套微信(企微端口就是配套企微)也要安装，指定版本的。
